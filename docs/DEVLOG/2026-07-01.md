# DEVLOG 2026-07-01 — 미래 시뮬레이션 Monte Carlo 확률 밴드

## 배경 / 왜 이걸 골랐나

이번 세션 후보 4개(환율 연말종가·Monte Carlo·IRP 추가납입 과세분리·배포) 중
**Monte Carlo 확률 밴드**를 선택. 근거:

- 환율 연말종가(ECOS)·배포(도메인·EC2)는 **외부 블로커**(인증키/사용자 액션) → 이번 세션에서 완결 불가.
- REVIEW Top10 **#4 "미래 '평균' 직선 = 확정수익 오인"**이 아직 남은 최대 신뢰성 구멍.
  지난 세션엔 경고 배너 + 과거빈도%로 **완화만** 했고, 로드맵의 "고도화: Monte Carlo / 변동성 밴드"가 근본 해법.
- 프론트/calc 순수영역 → 외부 의존 0, 결정적(seeded)이라 테스트 용이.

## 한 일

### calc.ts — `monteCarloBands()` (순수·결정적)
- **부트스트랩**: 선택 지수의 **실제 연도별 수익률**(로컬통화, 환율 미적용 — 미래모드 일관성)을
  매년 IID로 무작위 재추출. 각 draw를 `netBlendReturn(g, dep, cost)`로 안전 30/위험 70 + 실부담비용 블렌딩.
- **1,500 경로 × 전체 기간**(평균 시나리오처럼 30년 전 구간 투영 가능; 최악과 달리 이력 길이에 안 묶임).
- 연도별 **p10/p25/p50/p75/p90** 잔액 경로 + `probDcWins`(종말 DC≥DB 비율) + 중앙값 CAGR + 종말 10~90% 범위.
- **결정성**: `mulberry32` 고정시드(0x9e3779b9) → 리렌더/드래그마다 밴드 안정, SSR/CSR 좌표 불일치 없음, 테스트 박제 가능.
- 이력 < 5년이면 `null`(정직한 부트스트랩 불가). `SimResult.band?` 추가.
- `simulate()`에 `"band"` 분기: **중앙값 경로가 기존 verdict/세금 UI를 그대로 구동**(dcArr=p50, dcFinal=중앙값→퇴직소득세·IRP 재사용). band=null이면 평균 라인으로 폴백.

### Simulator.tsx / .module.css
- 미래 시나리오 3버튼 **평균 / 확률 밴드 / 최악**, **확률 밴드를 미래모드 기본값**으로.
- 차트: **10~90% 외곽 띠 + 25~75% 내곽 띠**(불확실성 콘) + 중앙값 DC선 + DB선. 밴드일 때 기존 단일 area는 숨김.
- 결과 요약: **"N% 확률로 DC가 DB보다 많습니다 · 10~90% 구간 A ~ B"**.
- 중립 톤 방법 캡션(과거 M년치·1,500회·매년 독립추출·미래 보장 아님). 범례에 `10~90%` 스와치.
- 호버 툴팁: 밴드모드에서 **DB / DC 중앙값 / 10~90% 범위** 3줄(박스 폭·높이 확장).
- 모바일 토글 3열(min 320px). `shareState`에 `scn=band` 허용.

### 테스트 (calc.test.ts) — 35 → **42**
- monteCarloBands: 결정성(동일입력 deep-equal)·이력부족 null·백분위 정렬(p10≤…≤p90)·0 앵커·확률 [0,1]·강세 flat 100%/약세 0%.
- simulate band: 전체기간 투영·band 존재·중앙값이 finals 구동·이력부족 폴백.

- `npx tsc --noEmit` 클린 · `npx vitest run` **42/42** · `npm run build` 통과.

  모바일 375px **가로 오버플로 0**, 다크모드 정상.

## 알아둘 점 (다음 세션 혼동 방지)
- **밴드 중앙값 CAGR(7.3%) > 평균 시나리오(6.3%)는 버그 아님.** 평균은 지수의 *전체 변동성이 반영된
  기하 CAGR*을 70% 슬리브에 상수 적용. 밴드는 연도별 수익률을 재추출·복리하므로 **안전 30%가 블렌드
  포트폴리오의 변동성 드래그를 줄여** 실현 기하수익이 더 높게 나옴. 둘 다 내부적으로 정합하며 각각 추정 라벨.
- IID(연도 독립) 가정이라 실제 평균회귀 대비 **밴드가 다소 넓을 수 있음**(보수적·정직한 방향). 캡션에 명시.
