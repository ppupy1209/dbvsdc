### (B) 결과 화면 (코드 변경 적음·고효과)
- **손익분기 수익률** `breakevenIndexReturn()`(이분법): "선택 지수가 연 X% 이상이면 DC, 미만이면 DB" — 사이트 정체성인데 화면에 없던 것을 박스로 노출. 과거 CAGR과 대조 표기.
- **세후 결론(IRP)** 요약 라인: 세전 verdict 외에 세후 실수령 우위 명시.
- 미래 '평균' 시나리오 **경고 배너**("매년 같은 수익률 가정 · 확정 수익 아님").
- 결과 하단 **인라인 면책**(투자권유·세무자문 아님).
- "연평균 수익률" 명칭 **분리**: 지수 카드=「지수 연평균(총수익·비용 전)」 vs 결과=「내 포트폴리오 연평균(안전30+위험70·비용 반영)」.
- "주요 기업"→「구성 종목 예시」(투자권유 오인 완화).

### (C) 환율·연도별 금리 — 메커니즘 + 근사데이터(사용자 결정: "명시")
- **연도별 안전자산 금리** `DEPOSIT_RATE_BY_YEAR`(1995~2025, 정기예금 근사). 백테스트는 연도별, 미래는 3% 고정. flat 3%의 과거 왜곡 제거.
- **원화 환산(FX) 토글** `USDKRW_RETURN_BY_YEAR`+`CURRENCY`+`isUsdIndex`. USD 지수·백테스트에서만 노출. KRW 환산 = (1+지수)(1+환율)−1. 미래/국내지수는 미적용.

## 후속 3 (결과 공유 URL + PDF)

- **공유 URL:** `web/src/lib/shareState.ts` `encodeShare/decodeShare` — 상태(mode·index·기간·연봉·상승률·시나리오·krw)를 쿼리스트링으로. **decode는 방어적**(미지값 드롭·범위 clamp). 단위테스트가 실버그 발견·수정: `Number(null)===0`이라 누락 파라미터가 0으로 → `p.has`/공백 가드. 테스트 5개(총 35).
- Simulator: 마운트 시 URL→상태 하이드레이트(client-only, SSR mismatch 회피), 상태 변경 시 `history.replaceState`로 주소창 동기화 → 현재 화면 자체가 공유 링크. "결과 링크 복사" 버튼(`navigator.clipboard`, 실패 시 URL 동기화 폴백).
- **PDF/인쇄:** "PDF로 저장·인쇄" 버튼 = `window.print()`. `@media print`(globals.css): 라이트 팔레트 강제(다크에서도)·`header` 숨김·`.noPrint`(버튼) 숨김·그림자 제거·`@page` 여백. 의존성 0(라이브러리 X).
- 검증: 하이드레이트(fwd·nq·20년·8000·7%·평균 전부 복원), URL 동기화(idx 클릭 반영) 프리뷰 확인. 클립보드는 프리뷰 eval-click이 신뢰 제스처 아니라 거부(실사용 정상). build 통과.

## 후속 4 (공식 데이터: World Bank 금리·환율 + KRX TR 절차)

요청: "BOK ECOS 공식 금리·환율, KRX 공식 TR". 환경 제약(Bash 외부망 차단 HTTP 000, WebFetch는 World Bank API에서 간헐 타임아웃)을 뚫고 **검증 가능한 공식값만 반영**.
- **예금금리·환율 → World Bank 공식 시계열로 교체**(손추정 대체):
  - `DEPOSIT_RATE_BY_YEAR` = World Bank **FR.INR.DPST**(예금금리, 연평균) 1996~2024. 1995(WB null)·2025(미발표) 잠정.
  - `USDKRW_RETURN_BY_YEAR` = World Bank **PA.NUS.FCRF**(원/달러 연평균)의 전년대비 변동 1995~2024. 2025 잠정.
  - ⚠️ **방법론 명시**: 환율이 *연평균* 기준이라 연말 급변동(2008·2022 말)은 일부만 반영 → Footer·주석·DATA-SOURCES에 고지. 연말종가(BOK ECOS 731Y001) 정밀화는 잔여.
- **재현 스크립트** `web/scripts/fetch-official-rates.mjs`: 네트워크 환경에서 `node`로 실행 → World Bank 풀링 후 붙여넣기용 배열 출력(추정 금지·재현 가능).
- **KRX 공식 gross TR(ks·kq)**: OTP 게이트라 자동수집 불가(확인 — 공개 JSON 403). `DATA-SOURCES.md`에 절차 명시. 근사 유지.
- **(추가) KRX 자동화 스크립트** `web/scripts/fetch-krx-tr.py`: `pykrx`가 OTP 핸드셰이크 내부 처리 → "수동 OTP 손작업" 제거. 사용자가 네트워크 PC에서 `pip install pykrx && python ...` 실행 시 코스피200 TR·코스닥150 TR 연간 수익률 배열 출력(붙여넣기용). 샌드박스는 외부망 차단이라 여기선 실행 불가. TR 반영 시 배당 가산(+1.8) 제거 안내 포함.
- 검증: FX ON 시 공식값 반영 확인(최악의 해 −11.8→−4.5%, 2022 원화 약세 완충), tsc/test 35/build 통과, 콘솔 0.
