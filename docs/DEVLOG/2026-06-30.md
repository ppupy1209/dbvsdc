- `api.ts` `auditMarketData()`: dev 모드에서 데이터 진입 시 콘솔 경고/에러(잘못된 편집이 조용히 배포되는 것 방지).
- `web/src/lib/__tests__/calc.test.ts` + vitest 도입(`npm test`). **27개** — 손익분기·FX·연도별 금리 포함. 퇴직소득세 1억/10년=426.25만, IRP 15년=×2/3 등 **손계산 박제**.

### (B) 결과 화면 (코드 변경 적음·고효과)
- **손익분기 수익률** `breakevenIndexReturn()`(이분법): "선택 지수가 연 X% 이상이면 DC, 미만이면 DB" — 사이트 정체성인데 화면에 없던 것을 박스로 노출. 과거 CAGR과 대조 표기.
- **세후 결론(IRP)** 요약 라인: 세전 verdict 외에 세후 실수령 우위 명시.
- 미래 '평균' 시나리오 **경고 배너**("매년 같은 수익률 가정 · 확정 수익 아님").
- 결과 하단 **인라인 면책**(투자권유·세무자문 아님).
- "연평균 수익률" 명칭 **분리**: 지수 카드=「지수 연평균(총수익·비용 전)」 vs 결과=「내 포트폴리오 연평균(안전30+위험70·비용 반영)」.
- "주요 기업"→「구성 종목 예시」(투자권유 오인 완화).

### (C) 환율·연도별 금리 — 메커니즘 + 근사데이터(사용자 결정: "명시")
- **연도별 안전자산 금리** `DEPOSIT_RATE_BY_YEAR`(1995~2025, 정기예금 근사). 백테스트는 연도별, 미래는 3% 고정. flat 3%의 과거 왜곡 제거.
- **원화 환산(FX) 토글** `USDKRW_RETURN_BY_YEAR`+`CURRENCY`+`isUsdIndex`. USD 지수·백테스트에서만 노출. KRW 환산 = (1+지수)(1+환율)−1. 미래/국내지수는 미적용.
  - 검증: S&P 백테스트 환산 ON → CAGR 10.7→11.7%, **최악의 해 −11.8→−8.1%**(2022 폭락을 달러 강세가 완충 = 환노출 ETF 실제 거동).
- 둘 다 **추정·검증 필요** 라벨(기존 배당/비용 근사와 동일 원칙). Footer 고지 갱신.

## 후속 (#8 입력 명확화 + #10 UX)

데이터 의존 없는 코드 달성분만 이어서 처리:
- **입력 명확화(#8):** "현재 연봉" 툴팁 = 세전 연 임금총액(상여 포함). DB 평균임금·DC 부담금이 **같은 base**라 상여 누락 시 둘이 함께 과소(비교/손익분기는 base에 거의 불변, 절대액·누진세만 영향) — 구조 변경 대신 라벨/툴팁로 정정.
- **용어 툴팁(#10):** 재사용 컴포넌트 `Term`(`web/src/components/Term.tsx`) — 점선 밑줄 + hover **및 탭/포커스** 표시(모바일엔 hover 없음). 7종 적용: 연봉·연봉상승률·지수 연평균(총수익/CAGR)·실부담비용·손익분기·IRP 연금수령(×2). 좌측 정렬 버블로 화면 밖 잘림 방지.
- **모바일 차트 터치(#10):** `onTouchStart/Move/End` 추가. `onMove`를 `setHoverFromClientX(clientX)`로 리팩터해 마우스/터치 공유. 합성 TouchEvent로 연도별 툴팁 노출 확인.
- **모바일 오버플로(#10):** `@media(max-width:560px)`에서 `.row` gap·`.rowLabel`·`.rowVal` 축소 → `.rowVal` 3px 오버플로/가로 스크롤 제거(offender 0).
