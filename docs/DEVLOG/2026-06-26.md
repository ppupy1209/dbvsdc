사용자 피드백 반영해 프론트를 여러 차례 다듬음. 모든 변경은 프리뷰(dev 서버 3100)에서 DOM 측정·동작 검증 후 도커(3000) 반영. 커밋 `6f798d1`.

## 수익률 모델 — 가격수익률 → 배당 재투자 포함(총수익)

- "지수 상승률 ≠ 배당 재투자 포함 수익률" 지적 반영.
- 연도별 실제 배당 31년×5지수를 정확히 재현 불가 → **지수별 장기 평균 배당수익률을 연 고정으로 가산**해 총수익 근사.
  - `indexData.ts` `DIVIDEND_YIELD`: sp 1.8 / nq 0.8 / dj 2.1 / ks 1.8 / kq 0.6 (%).
  - 적용 위치: `calc.ts` `cagrOf`, `simulate`(back 모드 연도별 + fwd 모드 CAGR). `totalReturn ≈ priceReturn + divYield`.
  - 원본 `RETURNS`(가격 등락률)는 그대로 두고 계산 단계에서 가산 → 추후 실제 데이터 교체 용이.
- 고지: 푸터 유의사항에 "가격 등락률 + 평균 배당수익률 = 배당 재투자 포함(총수익), 배당은 연 고정 가정" 명시.
- 검증: S&P 연평균 11.0%, 호버 2019년 DC 수익률 +22.4%(=0.3·3% + 0.7·(28.9%+1.8%)) 일치.

## 시뮬레이터 기능

- **미래 시뮬레이션 모드: 역전 시점 표시 제거** (역전 마커를 `mode === "back"`에만 렌더). 미래는 단일 가정 수익률이라 역전 의미가 없음.
- **연봉 상승률 슬라이더 max 10% → 15%.**
- **임금피크제 가정 옵션 제거** (UI 버튼·상태 삭제, `simulate` opts는 미사용으로 잔존). 푸터 문구의 "(옵션으로 가정 가능)"도 제거.
- **과거 백테스트 호버 툴팁에 해당 연도 DC 수익률 추가** (`SimResult.rets[]` 노출, 부호·색 구분). 미래 모드에선 미표시.
- **지수 선택 시 연평균 수익률 표시** — 지수 설명 박스에 "연 X.X% · 1995~2025 · 배당 재투자 포함" 행 추가(`cagrOf`).

## 테마 — 다크/라이트 수동 토글

- 우측 상단 내비 링크(시뮬레이터·IRP) 제거 → **해/달 슬라이드 토글**(`ThemeToggle`, 클라이언트 컴포넌트).
- `globals.css`: `prefers-color-scheme` 단독 → `data-theme` 기반(수동 토글) + 미선택 시 시스템 따름. 다크 토큰 2블록 유지.
- `layout.tsx`: 페인트 전 인라인 스크립트로 `data-theme` 선적용(FOUC 방지) + `suppressHydrationWarning`. `localStorage` 저장.
- 전환 시 `* { transition: background/border/color 0.28s }`로 스르륵, `prefers-reduced-motion` 시 해제.
- 다크 옅은 글씨 대비 상향: `--text-tertiary` #74747b→#98989f, `--text-secondary` #a6a6ad→#b8b8c0 (WCAG AA 통과).

- '미래 예측' → **'미래 시뮬레이션'** (탭·기간 라벨·결과 헤딩·푸터·메타데이터 일괄). "예측"은 단정적이라 면책 톤에 불리.

## 2026-06-26 — 금융 QA 보수화 수정

- 수익률 데이터 계약을 `gross_total_return`(배당 재투자 포함·현지통화·비용 차감 전)로 명시하고, `returnBasis/currency/dividendIncluded/expenseIncluded` 메타데이터를 API 응답에 추가했다.
- Spring 샘플 수익률과 프론트 샘플 수익률을 동일한 배열로 맞췄다. API 전환만으로 시뮬레이션 결과가 달라지는 문제를 제거하기 위함.
- DC 계산을 보수화했다.
  - 위험자산 비용 차감 0.70%p, 안전자산 비용 차감 0.10%p 반영.
  - 부담금은 연초 납입이 아니라 연말 납입으로 반영해 상승장 과대계상을 줄임.
  - 미래 모드는 전체 CAGR 반복 대신 동일 기간 과거 구간 중 최악의 DC 잔액 경로를 사용.
- 퇴직소득세에는 지방소득세 10%를 포함했고, IRP 연금수령은 15년 균등수령 기준으로 10년차까지 70%, 11년차부터 60% 과세를 반영했다.
- 작업자용 수정 사유 문서: `docs/FINANCIAL-QA-FIX-RATIONALE.md`.
