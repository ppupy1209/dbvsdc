# DATA-SOURCES — 외부 데이터 소스

DC형 투자상품/수익률 등 외부 데이터를 받아오는 소스 목록. **실시간 호출 대신 일배치 캐싱**이 원칙(이유: [ARCHITECTURE.md](ARCHITECTURE.md)).

> ⚠️ "예상 수익률"을 주는 API는 없다. 받아올 수 있는 건 **과거 실적/기준 데이터**뿐. 미래 수익률은 사용자가 슬라이더로 가정.

## 후보 API 목록

| # | 소스 | 제공 데이터 | 키 발급 | 비고 |
|---|---|---|---|---|
| 1 | 공공데이터포털 - 금융위 **펀드상품기본정보** | 펀드 표준코드/명칭/운용사 | data.go.kr 가입 후 활용신청 | [링크](https://www.data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15094792) |

## 사용 전략

- **MVP 단계**: 실시간 연동 없이, 대표 통계(원리금보장형 평균 vs 실적배당형 평균 과거 수익률)를 수동 정리한 정적 데이터로 시작. 사용자는 수익률을 직접 가정.
- **2차**: 위 #1·#2 (공공데이터포털, 키 발급 쉬움)부터 일배치 연동 → MySQL 캐싱.

## 키/시크릿 관리 규칙

- API 키는 **절대 git에 커밋하지 않는다.** 백엔드 환경변수(NCP)로 주입.
- 키 발급처·발급일·호출 제한(쿼터)을 이 문서 표에 기록해 둘 것.
