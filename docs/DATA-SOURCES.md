# DATA-SOURCES — 데이터 소스

> ⚠️ "예상 수익률"을 주는 API는 없다. 받는 건 **과거 실적/기준 데이터**뿐. 미래 모드는 과거 CAGR을 쓰는 평균 시나리오와 과거 최악 동일기간 경로를 쓰는 최악 시나리오로 나뉜다.

## ★ 핵심 결정 (2026-06-24): 지수 연간 수익률을 큐레이션해 보유 — API 키 불필요

백테스트 핵심 데이터는 **각 지수(S&P500·나스닥100·다우존스30·코스피200·코스닥150)의 연간 등락률**이다.
- ETF 시세 API로 실시간 수집하는 방식은 **폐기**. 이유: ETF는 상장 역사가 짧고(2020 전후), 환율·API 키·일배치 운영 부담이 큼.
- 대신 **지수 연간 총수익률 근사치(배당 재투자 포함·현지통화·비용 미반영)** 를 정적으로 보유. 지수별 보유 연도가 다르며, 키·배치 불필요. 매년 1행만 추가.
- 보유 위치: 프론트 `web/src/lib/indexData.ts` (단일 진실원), 백엔드는 동일 데이터를 MySQL에 시드해 `live`로 서빙.
- 출처: S&P500 = 연간 총수익률, 나스닥100/다우존스30 = 가격수익률 + 평균 배당수익률 근사, 코스피200 = 가격수익률 + 평균 배당수익률 근사, 코스닥150 = KODEX 코스닥150 ETF 조정종가 프록시. KRX TR/지수 직접 데이터로 정밀화 예정.
- ⚠️ 해외 지수는 USD 기준 → 원화 환산 옵션(백테스트) 제공. 환율 출처는 아래 "안전자산 금리·환율" 참고.

## 안전자산 금리·환율 — World Bank 공식값 반영 (2026-06-30)

연도별 예금금리·USD/KRW를 **World Bank Open Data 공식 시계열**로 교체(기존 손추정 대체). 재현 스크립트: [`web/scripts/fetch-official-rates.mjs`](../web/scripts/fetch-official-rates.mjs) — 네트워크 가용 환경에서 `node web/scripts/fetch-official-rates.mjs` 실행 시 `indexData.ts`에 붙여넣을 배열을 출력.

| 데이터 | 출처(indicator) | 기준 | 비고 |
|---|---|---|---|
| 안전자산 30% 금리 | World Bank **FR.INR.DPST** (Deposit interest rate %) | 연평균 | `DEPOSIT_RATE_BY_YEAR`. 1995(WB null)·2025(미발표)만 잠정. |
| 원화 환산 환율 | World Bank **PA.NUS.FCRF** (KRW/USD) | **연평균** | `USDKRW_RETURN_BY_YEAR`=전년대비 변동. 2025 잠정. |

- ⚠️ **방법론 한계**: 환율이 *연평균* 기준이라 연말 급변동(2008·2022 말 급등)은 일부만 반영. 정밀화하려면 **BOK ECOS 원/달러 종가(연말)** 로 교체 필요(`USDKRW` indicator 731Y001). API 키(ecos.bok.or.kr) 발급 후 연말종가 시계열로 대체.
- ⚠️ 금리도 ECOS *예금은행 정기예금(신규취급액, 1년)* 종가 기준으로의 정밀 교체는 잔여 과제(현재 World Bank 연평균으로 충분히 검증 가능).

## ★ KRX 공식 gross TR 수동 확보 절차 (ks·kq 잔여)

코스피200 TR·코스닥150 TR(배당 재투자 총수익지수)은 **KRX 고유 지수**라 공개 접근 불가(공개 JSON 엔드포인트 403, Yahoo엔 *가격*지수만, 다운로드는 OTP 토큰 흐름 필요). 자동/수동 두 경로:

### ① 자동 (권장) — OTP·로그인 손작업 없음
[`web/scripts/fetch-krx-tr.py`](../web/scripts/fetch-krx-tr.py). KRX의 공개 JSON 엔드포인트를 `requests`로 직접 호출한다(올바른 Referer 헤더만 있으면 로그인 불필요; pykrx는 최근 KRX 로그인 요구·Python 3.14 호환 문제로 폐기). 네트워크 되는 PC에서:
```
pip install requests
python web/scripts/fetch-krx-tr.py
```
출력된 `RETURNS.ks`/`RETURNS.kq`·`RETURN_YEARS`를 `indexData.ts`에 붙여넣고, **배당 가산 제거**(`DIVIDEND_YIELD.ks` 1.8→0; kq는 ETF 프록시 표기 제거). TR엔 배당이 이미 포함.
- ⚠️ 이 저장소 샌드박스는 외부망 차단이라 여기선 실행 불가 — 사용자 PC에서 실행. 스크립트는 **실패해도 진단(파인더 결과/응답 원문)을 출력**하니, 그 출력을 그대로 붙여주면 KRX 필드명/코드 매핑을 한 번에 교정 가능.

### ② 수동 (스크립트가 안 될 때)
1. [data.krx.co.kr](http://data.krx.co.kr) → **통계 → 지수 → 주가지수 → 개별지수 시세 추이**.
2. **코스피 200 TR** / **코스닥 150 TR** 선택, 기간 설정 → CSV 다운로드(브라우저가 OTP 처리).
3. 연말종가로 연간 수익률 = `종가_t / 종가_{t-1} − 1` → `RETURNS` 교체, 배당 가산 제거.

- 코스닥150 TR은 지수 기준일 2010-01-04 → 그 이전 불가. 코스피200 TR은 1990년대까지 존재(1995~2002 확장 가능).

## 수익률 기준 검증 (2026-06-29)

각 지수가 실제로 어떤 기준인지 점검한 결과:
- **S&P500**: 실측 연간 총수익률(배당 재투자). Slickcharts.
- **나스닥100 / 다우존스30**: ^NDX/^DJI 가격수익률(Yahoo로 확인 일치) + 고정 평균배당(+0.8 / +2.1). **추종 ETF(QQQ·DIA) 배당재투자 수익률과 대조하니 연도별 차이 대부분 ±0.5%p 이내** → 실측 TR 교체 가치 낮다고 판단(ETF는 보수 차감 net + 이력 짧음 + 프록시 기준 혼입). 현 근사 유지.
- **코스피200 / 코스닥150**: gross TR(코스피200 TR·코스닥150 TR)은 **KRX 고유 지수라 자동 수집 불가**(KRX OTP 게이트, 공개 사이트엔 가격지수만; TR ETF는 net·단기·프록시). 추정 금지 원칙에 따라 현 근사(코스피200=가격+1.8%, 코스닥150=KODEX ETF 조정종가 프록시) 유지. 진짜 gross TR은 KRX `data.krx.co.kr`에서 CSV 수동 다운로드 시 반영(TODO).

### 지수 이력 범위 — 왜 코스피200 2003~, 코스닥150 2016~ 인가 (2026-06-29)

- **우리 `ks`=코스피200, `kq`=코스닥150** (코스피/코스닥 *종합지수*가 아님). DC ETF(KODEX 200·KODEX 코스닥150)가 추종하는 게 종합이 아니라 200/150이라 그쪽으로 맞춤(d2255a6에서 종합→200/150 교체, 이력↓ 트레이드오프).
- **코스피200**: 기준일 1990-01-03, 발표 1994-06-15 → 1990년대부터 값 존재. 2010~만 갖고 있던 것은 큐레이션 한계였고, **2003~2025로 확장**(2003~2009 가격수익률 1stock1.com, 2010 값이 우리 데이터와 일치해 교차검증; +1.8% 배당 근사 동일 적용). 1995~2002는 자동수집 불가 → KRX 수동 CSV TODO.
- **코스닥150**: 기준일 **2010-01-04**(코스닥 종합 1996과 무관, 2010년에 생긴 지수). 선물 상장 2015-11, KODEX 코스닥150 ETF 2015 상장 → 우리 데이터(ETF 조정종가 프록시)는 2016~만 가능. 지수 자체(2011~2015) 또는 그 이전은 자동수집 불가 → KRX 수동 CSV 시 2011~까지 확장 가능(2010 이전은 지수 부재로 불가).

## DC 위험자산 비용 (실부담비용률, 2026-06)

위험자산(70%)에 지수별 대표 ETF의 **실부담비용률**(보수+매매·중개비 등, 명목 총보수 아님)을 차감. `web/src/lib/indexData.ts`의 `INDEX_COST`.

| 지수 | 대표 ETF | 실부담비용률(연) | 검증 |
|---|---|---|---|
| S&P500 | TIGER 미국S&P500 | 0.14% | ✅ |
| 나스닥100 | TIGER 미국나스닥100 | 0.15% | ✅ |
| 다우존스30 | TIGER 미국다우존스30 | 0.18% | ⚠️ 추정(미국ETF 대표값, 개별 미검증) |
| 코스피200 | KODEX 200 | 0.19% | ✅ |
| 코스닥150 | KODEX 코스닥150 | 0.32% | ✅ |

- 출처: 운용사 공시 / 실부담비용 비교(topetf.app 등). 대표값이며 상품·시점에 따라 변동 → 편집 가능한 기본값.
- ⚠️ 환율·환헤지비용은 미반영(환율 보류). 추적오차는 실부담비용에 일부 포함, 별도 분리 안 함.

## 사용 전략

- **MVP 단계**: 실시간 연동 없이, 지수 과거 수익률·대표 상품을 정적 데이터로 시작(현재 프론트 시안 = 예시값).
